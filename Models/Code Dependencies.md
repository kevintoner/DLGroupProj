## Code Dependencies 

The required library packages to install before running the notebooks.  
(The demos of each model require the same installations.)

**GRU_And_LSTM.ipynb**
* pip install matplotlib  
* pip install sklearn  
* pip install pandas  
* pip install torchvision  
* pip install torch  
* pip install numpy  

**NBEATS.ipynb**
* pip install matplotlib
* pip install sklearn
* pip install pandas
* pip install torch
* pip install numpy
* pip install darts
